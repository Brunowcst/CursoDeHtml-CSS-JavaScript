Principais formatações

Negrito/Destaque

Para deixar um trecho em negrito, usar a tag <B>(não-semântica)
Exemplo: Este trecho está em negrito

Para deixar um trecho em Strong, usar a tag <strong>(semântica)
Exemplo: Este trecho está em Strong.

Itálico/Ênfase

Para deixar um trecho em negrito, usar a tag <i>(não-semântica)
Exemplo: Este trecho está em itálico.

Para deixar um trecho com ênfase, usar a tag <em>(semântica)
Exemplo: Este trecho está com ênfase.

Marcador de texto

Para marcar algum trecho, usar a tag <MARK>
Exemplo: Este trecho está marcado.

Podemos mudar a cor do marcador usando <mark style>
Exemplo: Este trecho está marcado de verde.

Tamanho do texto

Para deixar algum trecho de caracteres pequenas, usar a tag <small>
Exemplo: Este trecho está pequeno.

Texto deletado

Podemos marcar um texto como deletado para indicar que ele pode ser lido, mas não considerado. Usar a tag <del>
Exemplo: Este trecho foi deletado.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado posteriormente. Usar a tag <ins>
Exemplo: Este trecho foi inserido.

Para sublinhar um texto, podemos usar a tag <U>, mas ela não é semântica, diferentemente da <ins>
Exemplo: Este trecho está sublinhado.

Texto sobrescrito

Usado para escrever coisas do tipo x^4 + 6, usar a tag<sup>
Exemplo: x4 + 6

Texto sobrescrito

Usado para escrever coisas do tipo H2O, usar a tag<sub>
Exemplo: H2O
