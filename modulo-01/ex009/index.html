Listas

Listas ordenadas

Exemplo de uma lista ordenada:

  1. 2 xícaras (chá) de açúcar
  2. 3 xícaras (chá) de farinha de trigo
  3. 4 colheres (sopa) de margarina
  4. 3 ovos
  5. 1 e 1/2 xícara (chá) de leite
  6. 1 colher (sopa) bem cheia de fermento em pó

Listas não-ordenadas

Exemplo de uma lista não-ordenada:

  • Padeiro
  • Mecânico
  • Engenheiro
  • Taxista
  • Presidente
  • Analista

Juntando listas

Usando uma lista ordenada dentro de outra:

  1. Brasil
    1. São Paulo
    2. Rio Grande do Sul
  2. Estados Unidos
    1. Califórnia
    2. Texas

Usando uma lista desordenada dentro de uma ordenada:

  1. Brasil
    • São Paulo
    • Rio Grande do Sul
  2. Estados Unidos
    • Califórnia
    • Texas

Também pode ser feito:

  1. Brasil
    • São Paulo
      • Santos
      • Guarulhos
    • Rio Grande do Sul
  2. Estados Unidos
    • Califórnia
    • Texas

Listas de defiição

HTML
HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.
CSS
Trata-se de uma linguagem de marcação,e representa diversas possibilidades para a formatação. O CSS ajuda a editar, alinhar, remover e trabalhar no espaço entre elementos de uma página.
JavaScript
é uma linguagem de programação que permite a você implementar itens complexos em páginas web.
